Projekt & Teknisk- beskrivning:
Vi har utvecklat en hemsida som hjälper elever att plugga och hålla koll på uppgifter som prov eller deadlines. Vi har även en pomodorotimer som hjälper eleverna att plugga och även ta pauser för att effektivisera arbetet så bra som möjligt. Förutom en pomodorotimer har vi även en chatbot som kan svara på frågor som användarna har. För att öka användarupplevelsen så mycket som möjligt har vi inte bara lagt in behagliga färger men även att man kan få välja färg själv på sina händelser i kalendern efter sina egna preferenser. Vi har stött på några problem under kodningen som att fixa så att tiderna i kalendern skulle följa med rutorna som visar händelserna. Vi hittade sedan att tidkolumnen låg utanför våran scrollarea i css koden som då gjorde att den inte hängde med upp med de andra kolumnerna som låg innanför den. När vi lade in den så fungerade det och den passar perfekt med de andra kolumnerna. Vi hade även problem med att kalendern låg en dag före hela tiden och det löste vi genom att ändra kodraden som visade vilken som var den första dagen på året och månaden från att det stod +1 till -0. Vi behövde skriva minus 0 eftersom -1 blev en dag innan och om vi skrev +1 eller bara tog bort det helt blev det en dag efter. Därför behövde vi skriva -0.
